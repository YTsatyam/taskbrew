# TaskBrew ☕
A sleek to-do list web app built using pure HTML, CSS, and JavaScript.

## 🔧 Features
- Add, edit, delete tasks
- Filter: All | Active | Completed
- Priority labels (Low, Medium, High)
- Dark / Light mode toggle
- Persistent using `localStorage`
